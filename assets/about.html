OSPedia is an open-source website that aims to educate people about the history of various influential operating systems of the present, including Windows and macOS. It also aims to make people appreciate today's software and make information about legacy operating systems easily accessible to any one who may need it without having to dig for it throughout the depths of the internet.

You can view the source code for this website on GitHub. Please contribute! The CONTRIBUTING file should be comprehensive enough to assist you if you do plan on helping us!
